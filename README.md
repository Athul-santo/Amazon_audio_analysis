# 🎧 Amazon Audio Product Analysis 

## 📌 Overview
Analyzed Amazon audio product data using EDA, machine learning models, and Power BI dashboards to generate insights and improve product recommendations.

## 🛠️ Tools
Python, Power BI

## ⚙️ Work Done
- Performed **data cleaning and EDA** to analyze price, ratings, and reviews  
- Built multiple models including:
  - **Device Prediction Model**
  - **Recommendation System**
  - **Clustering Model** (product segmentation)  
- Developed an **interactive Power BI dashboard** for visualization  

## 📈 Results
- Identified key patterns in product data  
- Segmented products into meaningful groups  
- Improved recommendation approach using data-driven insights
